Dictionary App

This project utilized a Vite React template to create a dictionary app using a dictionary API. Users can search for words and obtain the phonetics, definitions and examples. The dictionary consists of a simple UI layout using CSS. The tech stack is React, Axios (Fetching the API), Dictionary API, CSS and it consists of a responsive layout.
